# Global Layoffs Trends Analysis (2020-2025)

## Executive Summary
This project provides a comprehensive analysis of global workforce reductions during the post-pandemic era (2020-2025). By processing large-scale layoff data, this study identifies industry-specific trends, geographic hotspots, and the correlation between tech-sector volatility and macro-economic shifts.

## Key Objectives
*   **Data Integrity:** Clean and standardize raw layoff data using advanced SQL techniques (CTEs, Window Functions).
*   **Trend Identification:** Quantify the acceleration of layoffs in the tech sector vs. traditional industries.
*   **Actionable Insights:** Provide a longitudinal view of workforce changes to assist in labor market forecasting.

## Methodology
The project utilizes a **Dual-Toolchain approach**:
1.  **SQL (Data Engineering):** Primary cleaning, deduplication, and aggregation performed in SQL. Key script: `LayoffsSQL Script .sql`.
2.  **R (Exploratory Data Analysis):** Statistical analysis and visualization of time-series data. Key script: `World Layoffs 2023-25.Rmd`.
3.  **Visualization:** High-impact charts generated in R (`ggplot2`) and documented in the portfolio report.

## Tools & Technologies
*   **Database:** SQL (Standard SQL / MySQL compatible).
*   **Analytics:** R (tidyverse, lubridate).
*   **Reporting:** RMarkdown, PDF Portfolio Report.
*   **Visualization:** Tableau (Workbook included in project folder).

## Project Structure
*   `LayoffsSQL Script .sql`: The core data cleaning engine.
*   `World Layoffs 2023-25.Rmd`: The statistical analysis and visualization source.
*   `Global_Layoff_Trends_Portfolio_Report.pdf`: Final executive-style report.
*   `layoffs.csv`: The underlying dataset.
